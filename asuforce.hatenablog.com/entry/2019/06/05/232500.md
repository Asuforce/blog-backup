native camp 今日の学びは r, l の発音 r は喉の奥で下を巻く、 l は舌を下の歯にくっつけるように発音する。r, l は聞き取りづらいし発音も難しいけど、ポイントを教わるとなんとなく違いがわかるようになってきた。日々練習することをおすすめされた。

```txt
right / light
grass / glass
read / lead
road / load
arrive / alive
```
